This plugin provides insight into your content empowering smart site administrators.

== Description ==

The Content Analytics plugin provides insight into the digital content that your create.  The goal this plugin is to provide additional visibility into content being created, the nature of that content, summarization of the content, automatically generated content meta, content suggestions or recommendations, and how it is used.

== Benefits ==

Provides content analytics, such as:
* words
* segments (e.g., sentences)
* media (e.g., images, links, YouTube)
* keywords (e.g., most commonly used)
* detected languages
* sentiment
* content velocity

== Installation ==

Upload the Content Analytics plugin to your WordPress installation, Activate it, then your done!
